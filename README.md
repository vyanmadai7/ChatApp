## RealTime Chat App

This is a simple chat app that updates messages in real time. It is built with Django, AJAX, jQuery, and HTML/CSS.

## Features

Messages update automatically in real time

Create a new chat room or join an existing one

Simple and modern user interface

New messages scroll down automatically

Very lightweight   no WebSockets required

## How to Run
git clone https://github.com/vyanmadai7/ChatApp<br>
cd ChatApp<br>
pip install django<br>
python manage.py migrate<br>
python manage.py runserver

Then open your browser and go to:
http://127.0.0.1:8000/
