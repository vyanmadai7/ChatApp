## Real-Time Chat App

A lightweight chat application that updates messages instantly—no WebSockets needed. Built with Django, AJAX, jQuery, and HTML/CSS for a smooth, modern experience.

## Features

Real-time message updates

Create or join chat rooms easily

Clean and intuitive user interface

Auto-scrolls to new messages

Lightweight and fast

## Getting Started
git clone https://github.com/vyanmadai7/ChatApp
cd ChatApp
pip install django
python manage.py migrate
python manage.py runserver

Open your browser at:
http://127.0.0.1:8000/
