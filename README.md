## Real-Time Chat App

A lightweight chat application that updates messages instantly—no WebSockets needed. Built with Django, AJAX, jQuery, and HTML/CSS for a smooth, modern experience.

## Features

Real-time message updates<br>

Create or join chat rooms easily<br>

Clean and intuitive user interface<br>

Auto-scrolls to new messages<br>

Lightweight and fast

## Getting Started
git clone https://github.com/vyanmadai7/ChatApp<br>
cd ChatApp<br>
pip install django<br>
python manage.py migrate<br>
python manage.py runserver<br>

Open your browser at:<br>
http://127.0.0.1:8000/
